# Sample Account Manager REST API

## Basic Information
- Language: Java
- Build tool: Maven
- Spring Boot: 2.7.5
- Default database: H2
- Recommended IDE: JetBrains IntelliJ

## Quick Start
1. Compilation
```shell
mvn compile  
```

2. Run
```shell
mvn spring-boot:run
```
Swagger - http://localhost:8080/swagger-ui/index.html. 
You may execute ``data-h2.sql`` in ``resources`` to initialize the database.

3. Run Test
```shell
mvn test
```
